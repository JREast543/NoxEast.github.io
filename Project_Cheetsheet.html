HTML Table Reference

HTML Table Tags

Tag Name Description
<table> Table This tag defines an HTML table as it contains other HTML elements that determine the structure of the table.
<thead> Table Head This tag defines the column headers of the table and is child to the table element.
<tbody> Table Body This tag defines the rows containing the table data.
<tr> Table Row This tag defines the row container.
<td> Table Data This tag allows data to be contained in rows.
<tfoot> Table Foot This tag defines the footer in the table.

HTML Table Attributes

Attribute Name Description
rowspan Row Span This Attribute represents the number of rows to span.
colspan Column Span This atrribute represents the number of columns to span.
